# Joomla AI Translation Plugin (POC)
Project Overview
This is a Proof of Concept (POC) developed as part of my GSoC 2026 application for the Joomla! project. The goal of this plugin is to demonstrate a basic integration of Joomla's event system with an automated translation workflow.

# Features
* **Event Triggering:** Listens to the `onContentAfterSave` event.
* **Context Awareness:** Ensures translations are only triggered for the `com_content`.
* **Foundation for RAG:** Starting point for implementing the RAG logic.
